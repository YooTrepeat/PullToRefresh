## 使用步骤
### 1. 添加依赖 ###
	//在项目下的build.gradle
	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}

	//在app模块下的build.gradle
	dependencies {
	        compile 'com.github.open-android:PullToRefresh:v1.0'
	}
